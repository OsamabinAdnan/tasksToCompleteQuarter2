Definition tag

Definition List Tag <dl>:

The <dl> tag defines the start of a definition list. It acts as a container for <dt> and <dd> tags, grouping terms and their definitions.

Definition Term Tag <dt>:

The <dt> tag represents a term in the definition list. It usually appears in bold or as a header for the definition following it.

Like

Information technology
It is a branch of computer science, defined as the study of procedures, structures, and the processing of various types of data.

Definition Description Tag <dd>:

The <dd> tag provides the description or details of the term defined in the preceding <dt> tag, typically indented.

Example

HTML
A standard markup language for creating web pages.
CSS
A language used to style HTML documents.
