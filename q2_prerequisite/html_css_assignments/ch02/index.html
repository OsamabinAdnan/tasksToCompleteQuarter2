Some More Tags

Bold and Strong Tags

Bold Tag

Bold tag <b> is the HTML element that enables text formatting. This tag defines bold text on a given page without giving any additional importance to the indicated content

Strong Tag

Strong tag <strong> is used to create a 'strong' element, which represents strong text in an HTML document

Italic and Emphasis tags

Italic tag

The <i> tag is used to apply italic styling to text, usually for stylistic purposes, such as to indicate a change in mood or for foreign words. It doesn't add semantic meaning but affects how the text looks.

Emphasis Tag

The <em>> tag is used to stress or emphasize text, which may change the meaning or add importance. It has semantic meaning, indicating that the text inside should be emphasized when read by screen readers. Typically, it also displays as italics but with added meaning for assistive technology.

Citation and Definition Tags

Citation Tag

The <cite> tag is used to mark the title of a work, such as a book, article, movie, or other creative work, often displaying it in italics. It provides semantic meaning, helping screen readers and search engines identify cited material.

The Great Gatsby, render as The Great Gatsby typically in italics.

Definition Tag

The <dfn> tag is used to specify the defining instance of a term or phrase within a document. It' often styled in italics by default, and screen readers may emphasize it, helping users understand that this is the term's definition.

Example: A Black Hole is a region of space from which nothing, not even light, can escape.

Insert and Delete tags

Insert Tag

The <ins> tag is used to show inserted or added text, often displayed with an underline. This tag is semantically meaningful, as it highlights additions in content, which is useful in collaborative editing or revision tracking.

Example: New text displays the text as underlined, indicating it was newly added.

Delete Tag

The <del> tag represents text that has been removed or deleted, often shown with a strikethrough. This is useful in tracking changes in documents or highlighting edits.

Example: Old text shows the text with a line through it, indicating it was deleted.

Strikethrough Tag

The <s> tag applies a strikethrough effect to text, typically indicating incorrect or outdated information. This tag is often used stylistically, but it lacks semantic meaning.

Example: Outdated information displays the text with a strikethrough.

Quotations Tags

Blockquote Tag

Used for longer quotations, this tag generally indents the quoted content, often with block-level formatting.

Example:

Long quote here.

Quote Tag

Used for shorter, inline quotes, typically displayed with quotation marks around the text.

Example: Inline quote appears with quotes around it.

Superscript and Subscript

Superscript Tag

The <sup> tag displays text as a superscript, raising it slightly above the normal line, often used for exponents or ordinal indicators.

Example: x2 shows as x².

Subscript Tag

The <sub> tag displays text as a subscript, lowering it below the normal line, commonly used for chemical formulas or mathematical notation.

Example: H2O shows as H₂O.

Address Tag

The <address> tag is used to provide contact information for the author or owner of a document or section. It usually appears in italics and may include email, phone numbers, or other contact details.

Example:

Contact: email@example.com
displays the contact information in a stylized format.
